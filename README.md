## Prerequisites:
1. <a href="https://nodejs.org/es/download/">Node.js</a> to be installed.
2. Angular cli:
```
$ npm install -g @angular/cli
```
3. Ionic cli:
```
$ npm install -g @ionic/cli
```

## How to use:
1. clone the project.
2. cd into the project.
3. install the dependencies:
```
$ npm install
```
4. run the project:
```
$ ionic serve
```
5. That's it, enjoy.
